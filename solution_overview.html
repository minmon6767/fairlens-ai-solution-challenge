⚙️ How It Works
Five simple steps from data to fair AI.
1

Data Ingestion

Upload model predictions CSV or connect via REST API. FairLens automatically detects protected attributes (gender, race, age) from column names.

2

Bias Analysis Engine

FastAPI backend runs 10+ fairness metrics in parallel using AIF360. Results are stored in Firestore with timestamped audit logs.

3

Gemini XAI Explanation

Gemini 1.5 Pro generates a plain-language bias report — what was biased, which groups were affected, severity, and root cause analysis.

4

Mitigation Recommendations

System proposes ranked fixes with confidence scores. User accepts or applies them directly via one-click re-training trigger.

5

Compliance Report Export

Download PDF/JSON audit trail for regulatory compliance — covers EU AI Act Article 10, EEOC Uniform Guidelines, and GDPR Article 22.
